# منصة مسابقة المرحلة الثانوية — نموذج جاهز لـ GitHub Pages

## التشغيل محليًا
1. افتح `teacher.html` لإضافة أسئلة.
2. افتح `student.html` لتجربة المسابقة.

## الرفع على GitHub Pages
1. أنشئ مستودع جديد على GitHub.
2. ارفع جميع الملفات في جذر المستودع.
3. من إعدادات المستودع (Settings) → Pages → Source = `main`، وFolder = `/(root)`.
4. احفظ. سينشأ رابط للموقع خلال دقائق.

> ملفات هذا النموذج تعمل بدون خادم وتستخدم LocalStorage.
